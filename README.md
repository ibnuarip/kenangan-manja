# Kopi Kenangan Manja

Website landing page sederhana dan elegan untuk kedai kopi "Kopi Kenangan Manja". Dibuat dengan HTML, CSS (menggunakan arsitektur variabel warna dan responsif flexbox/grid), dan sedikit sentuhan JavaScript.

## Fitur Utama

- **Desain Modern:** Memiliki estetika elegan dengan latar belakang gelap, aksen warna emas/cokelat, dan efek kaca buram (_glassmorphism_) pada bilah navigasi (navbar).
- **Responsive Layout:** Halaman ini dapat beradaptasi dan tampil apik mulai dari layar komputer, tablet, hingga ponsel pintar (_smartphone_).
- **Animasi Mulus:** Memiliki efek transisi warna dan transformasi interaktif yang menyempurnakan interaksi setiap saat ikon, tombol, maupun kartu produk diarahkan (_hovered_).
- **Modal Box Interaktif:** Dilengkapi dengan pop-up (modal) detail produk tanpa harus berpindah ke halaman lain.

## Struktur Folder Utama

- `index.html`: Berkas utama yang berisi struktur konten halaman.
- `css/style.css`: Berkas _styling_ yang mengatur tata letak, warna, responsivitas, dan animasi di dalam website.
- `js/script.js`: Berisi program logika dasar untuk fungsionalitas menu samping, bilah penelusuran (_search-bar_), keranjang belanja, hingga operasional pop-up modal detail produk.
